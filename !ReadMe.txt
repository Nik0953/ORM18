    1. вспомогательный модуль 'make_json_data.py'
перекачивает данные из старой базы 'data/vac_hh.db'
в папку 'data/old_data_json/' в формате json,
чтобы потом достать оттуда для новой базы.

      вспомогательная функция в модуле get_json_data.py


    2. модуль db_obj_declaration.py
Объявление всех классов данных,
которые будут использоваться в работе с базой данных


    3. модуль make_new_db.py
Модуль создает новую базу данных '/data/new_data/new_hh.db'
из таблиц json
с помощью ОRM sqlalchemy

Этот модуль надо однократно запустить, чтобы появилась новая база данных


    4. модуль prepare_web_info.py
    содержит функции, возвращаемые в функции Flask.

    5. модуль main.py
Запускается "сервер" для web-страниц запросов и их результатов.
